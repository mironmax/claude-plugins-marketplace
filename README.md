## Installation

### 1. Add This Marketplace

```
/plugin marketplace add mironmax/claude-plugins-marketplace
```

### 2. Install Plugins

```
/plugin install knowledge-graph@maxim-plugins
```

### 3. Restart Claude Code

The plugin will be available after restart.
